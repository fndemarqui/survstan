# Piecewise exponential baseline

This function is used to allow the user to specify the piecewise
exponential baseline with arbitrary time grid/number of intervals.

## Usage

``` r
piecewise(rho = NULL, m = NULL)
```

## Arguments

- rho:

  the specified time grid; default is NULL.

- m:

  the number of intervals; default is NULL.

## Value

a list with the baseline name, and the provided time grid and number of
intervals.
