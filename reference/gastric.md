# Gastric cancer data set

Data set from a clinical trial conducted by the Gastrointestinal Tumor
Study Group (GTSG) in 1982. The data set refers to the survival times of
patients with locally nonresectable gastric cancer. Patients were either
treated with chemotherapy combined with radiation or chemotherapy alone.

## Format

A data frame with 90 rows and 3 variables:

- time: survival times (in days)

- status: failure indicator (1 - failure; 0 - otherwise)

- trt: treatments (1 - chemotherapy + radiation; 0 - chemotherapy alone)

## References

Gastrointestinal Tumor Study Group. (1982) A Comparison of Combination
Chemotherapy and Combined Modality Therapy for Locally Advanced Gastric
Carcinoma. Cancer 49:1771-7.
